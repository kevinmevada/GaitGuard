Process

From raw sensor data
to clinical insight.

A fully automated five-stage pipeline transforms multi-sensor IMU recordings into structured gait intelligence within seconds.

01
IMU
IMU Ingestion
Head, lower back, left & right foot sensors sampled at 100 Hz with automatic timestamp synchronization.
02
Wave
Signal Processing
Butterworth bandpass filtering, drift correction, and automatic gait cycle segmentation.
03
Lab
Feature Extraction
41 biomechanical features per trial — including cadence, stride length, asymmetry index, and harmonic ratio.
04
Gear
Ensemble Models
XGBoost, LightGBM, Random Forest, and SVM combined through soft voting for robust pattern recognition.
05
Report
Structured Output
Quantitative gait scoring (0–100), SHAP-based feature attribution, and anomaly classification.

Anomaly Detection

Unsupervised outlier
detection. Always on.

Three unsupervised algorithms running in parallel - flagging abnormal gait patterns that fall outside learned normative bounds.

Forest
Isolation Forest
Random recursive partitioning isolates anomalies faster on high-dimensional gait feature spaces. Scales without density assumptions.
LOF
Local Outlier Factor
Density-based neighborhood comparison catches anomalies invisible to global thresholds - essential for cohort-specific edge cases.
SVM
One-Class SVM
Learns hypersphere boundary of normal gait - ideal for detecting novel pathological patterns not seen during training.
Vote
Ensemble Consensus
A trial is flagged anomalous only when ≥2 methods agree, providing a conservative internal review signal for follow-up analysis.
