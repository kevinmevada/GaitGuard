## Notes for manuscript text

- Several classic papers do **not** report AUC; forcing numeric AUC comparison can be misleading.
- The strongest direct comparison in this manuscript should emphasize **task differences** (fall detection vs fall-risk screening vs pathology-tier discrimination).
- If reviewers request strict one-to-one fall-risk AUC benchmarks, cite only studies with explicit ROC-AUC and similar endpoint definitions in the main text, and keep others as contextual background.
