## Institutional Review Board Statement

This study used a publicly available, de-identified dataset hosted on Figshare ([https://doi.org/10.6084/m9.figshare.28806086](https://doi.org/10.6084/m9.figshare.28806086); Voisard et al., *Scientific Data* 2025, [https://doi.org/10.1038/s41597-025-05959-w](https://doi.org/10.1038/s41597-025-05959-w)). The original data collection was approved by the **Comité de Protection des Personnes Île-de-France II** (CPP ID **2014-10-04 RNI**). Patients and healthy volunteers provided **written informed consent** to participate and to share data, as reported by the dataset authors. **No new human participants were recruited** and **no new human data were collected** for this secondary analysis; only previously published, de-identified gait signals and metadata were processed.

---

## Informed Consent Statement

Informed consent was obtained from all participants by the original investigators (Voisard et al., 2025) under the ethics approval above. No additional informed consent was required for this secondary computational study of de-identified data.

---

## Data availability (cross-reference)

The same dataset DOI should appear in the Data Availability Statement. This repository provides analysis code and model cards; it does not redistribute raw Figshare files.

## One-sentence version (abstract / cover letter)

We analyzed only the public de-identified IMU gait dataset of Voisard et al. (Figshare 10.6084/m9.figshare.28806086), originally approved by CPP Île-de-France II (2014-10-04 RNI), without collecting new human data.
